# Getting Started

Ask LLM is a modern command-line tool for calling multiple LLM APIs. It is built with [Typer](https://typer.tiangolo.com/), [Pydantic](https://docs.pydantic.dev/), [Rich](https://rich.readthedocs.io/), and [Loguru](https://loguru.readthedocs.io/).

## What You Can Do

- **Ask** — Send single prompts to an LLM with file or text input.
- **Chat** — Start an interactive session with history, meta commands, and shell integration.
- **Translate** — Translate Markdown, text, and Jupyter notebooks with intelligent chunking.
- **Batch** — Process multiple tasks across one or more models concurrently.
- **Paper** — Explain academic papers section by section using configurable prompts.
- **Format** — Normalize Markdown heading levels using an LLM.
- **Config** — Manage providers, test connections, and initialize configuration files.
